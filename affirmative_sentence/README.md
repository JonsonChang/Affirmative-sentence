# 肯定语句 Affirmative Sentence

一款帮助培养积极心态的跨平台移动应用，每日推送肯定语通知。

## 功能特点

- 每日定时推送肯定语通知
- 自定义肯定语内容和管理
- 个性化通知时间设置  
- 简洁友好的用户界面
- 支持Android和iOS平台

## 技术栈

- Flutter 跨平台框架
- SQLite 本地数据库
- 本地通知服务

## 运行项目

1. 确保已安装Flutter SDK
2. 克隆本项目
3. 运行 `flutter pub get` 安装依赖
4. 运行 `flutter run` 启动应用

## 项目结构

```
lib/
├── models/        # 数据模型
├── services/      # 业务逻辑
├── screens/       # 界面屏幕
└── widgets/       # 可复用组件
```
